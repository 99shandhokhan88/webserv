🔧 Come Testare

  1. Con index.html presente: Vai su / → Dovrebbe servire index.html
  2. Senza index.html: Rinomina index.html → Vai su / → Dovrebbe mostrare directory listing (se autoindex ON) o 403 (se autoindex OFF)
  3. Directory con autoindex ON: Vai su /uploads/ → Dovrebbe mostrare file listing
  4. Directory con autoindex OFF: Vai su /css/ → Dovrebbe restituire 403 Forbidden

Questa implementazione segue gli standard HTTP/1.1 e si comporta come Nginx.
