About Webserv

This is a custom web server built from scratch. It supports static files, CGI scripts, and custom error pages.

Features

  • Static file serving
  • CGI support (Python, PHP, etc.)
  • Custom error pages
  • File uploads
  • HTTP Methods: GET, POST, DELETE
